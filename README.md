# Daia's Life Tracker

## Deploy to Vercel (5 minutes, free)

### Step 1 — Create a GitHub account (if you don't have one)
Go to github.com and sign up free.

### Step 2 — Upload this folder to GitHub
1. Go to github.com/new
2. Name it `daia-life-tracker`, click "Create repository"
3. Click "uploading an existing file"
4. Drag and drop ALL files from this folder (keep the folder structure)
5. Click "Commit changes"

### Step 3 — Deploy to Vercel
1. Go to vercel.com and sign up with your GitHub account
2. Click "Add New Project"
3. Select your `daia-life-tracker` repository
4. Click "Deploy" — Vercel auto-detects Vite ✓
5. Done! You get a URL like: https://daia-life-tracker.vercel.app

### Step 4 — Add to your phone home screen
- iPhone: Open your Vercel URL in Safari → Share → "Add to Home Screen"
- Android: Open in Chrome → menu → "Add to Home Screen"

## Making changes
Come back to your Claude conversation and ask for changes.
Claude will give you an updated App.jsx — just replace the file in GitHub
and Vercel auto-redeploys in ~30 seconds.

## Files
- `src/App.jsx` — the entire app (this is the only file you'll ever need to replace)
- `src/main.jsx` — entry point (never needs changing)
- `src/index.css` — global styles (never needs changing)
- `index.html` — HTML shell (never needs changing)
- `package.json` — dependencies (never needs changing)
- `vite.config.js` — build config (never needs changing)
- `tailwind.config.js` — styling config (never needs changing)
